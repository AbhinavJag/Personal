Methodology

How this
was built.

This platform synthesizes six federal data sources into a unified analysis of American household economic security from 2000 to 2024.

Real Wage Calculation
Nominal wages deflated using BLS CPI-U (1982–84 = 100). Real Wage = Nominal Wage × (CPI₂₀₀₀ / CPIₜ). Base year: 2000.
Housing Affordability Ratio
Price-to-Income (PTI) = Median Home Price ÷ Median Household Income. Standard metric used by IMF and OECD for cross-national comparison.
Economic Security Index (ESI)
Composite weighted index: 35% real wage, −25% PTI deviation, −20% rent burden deviation, −12% healthcare index, −8% tuition index. Normalized to 100 in 2000.

Data Sources

  • BLS CES — Average hourly earnings by sector, 2000–2024
  • BLS CPI-U — Consumer Price Index, all urban consumers
  • U.S. Census ACS — Median household income; median gross rent
  • NAR — Median existing home sale price
  • CMS — National health expenditure per capita
  • NCES — Average tuition & fees, public 4-year institutions
  • U.S. DOL — Federal minimum wage history
Research Paper
This platform accompanies a full academic paper: "The Widening Gap: Wage Stagnation, Inflation, and the Erosion of American Economic Security (2000–2024)" — available for download from the GitHub repository. Includes methodology, statistical analysis, literature review, and policy implications.
Reproduce this analysis
All data and Python analysis code (economic_analysis.py) are open source. The script collects data from public APIs, computes all derived metrics, and generates publication-quality charts using pandas, NumPy, scipy, and matplotlib.
